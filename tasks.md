## 🛠 详细任务清单

### 1. 多开发工作空间管理 (Workspace Management)
- [x] **服务器配置**：支持添加/管理多个远程开发节点，包括 SSH 主机、端口、用户名和认证方式（密码或密钥）。
- [x] **一键连接控制**：前端支持“启动 Gateway”按钮，后台建立 SSH 通道并向前端推送连接生命周期事件。
- [x] **网络状态感知**：实时展示工作区当前的连接状态（已连接、握手、断开）以及往返网络延迟 (Latency Ms)。

### 2. 本地端口映射 (SSH Tunnel Port Forwarding)
- [x] **双向端口映射**：本地 TCP 端口（如 5173, 8080）与远程服务器端口的一键安全映射与隧道穿透。
- [x] **实时带宽流速图表 (Throughput)**：使用 Recharts 绘制高精度时序图表，展示数据发送（Upload）与接收（Download）的实时吞吐量。
- [x] **高性能零内存拷贝优化**：在 Go 核心网关底层使用 `io.Copy` 与内核缓冲区映射实现高效、低开销的双向数据搬运，在不佳网络下确保极低包重传与稳定性。

### 3. 文件同步视觉服务 (File Watcher & Delta Sync)
- [x] **本地热重载监听 (`fswatch`)**：低 CPU 开销的文件系统轮询/侦听机制，当检测到本地文件被修改（或通过“模拟修改”测试触发）时立即触发同步流程。
- [x] **SHA-256 增量同步算法**：比对两端文件的哈希校验码，计算差异增量 (Delta Sync)，只传输变化的内容，避免全量上传。
- [x] **可视化文件操作控制台**：
  - [x] 提供直观的本地开发目录与远程服务器目录的树形结构对比界面。
  - [x] 底部配备高可读性的实时同步操作日志。

### 4. 架构优化 AI 助手 (AI Architecture Copilot)
- [x] **源码智能阅读器**：AI 助手能直接读取并解析 `/go-gateway` 目录下的核心 Go 文件源码（如 `main.go`, `forward.go`, `sync.go`）。
- [x] **架构调优交互**：集成专门的 Gemini 模型对话引擎，支持探讨多线程 TCP 并发、Byte Pool 缓冲池扩展、多平台交叉编译等底层高性能网络设计。

### 5. ✨ 工作区 Web 终端仿真器 (Web Terminal - 额外扩展)
- [x] **Xterm.js 终端集成**：基于高成熟度 `xterm` 与 `xterm-addon-fit` 库，在“工作区终端”新标签页提供高保真终端。
- [x] **WebSocket 全双工管道**：Express 服务端启动 `/api/terminal` websocket，接收前端击键/控制序列，并拉起系统 Shell (`bash`/`sh`) 实现双向实时输入输出流对接。
- [x] **终端多主题选择器 (Theme Selector)**：
  - [x] 支持在菜单中一键切换 Dracula, Solarized Dark, Gruvbox Dark, Nord, Monokai, Cosmic Dark 等多款经典终端主题。
  - [x] 自动将用户的主题偏好持久化保存在 `localStorage` 中，后续进入自动保持偏好状态。
- [x] **多维自适应缩放**：监听窗口 resize 事件，实时向后台发送列数/行数 resize 信息，保证排版与交互的最佳适配。

## 🚀 后续技术演进方向 (Future Roadmap)
1. **多路复用 SSH 会话优化**：在单一 SSH 物理连接上利用多通道（Mux）同时运行 Terminal、端口转发以及文件同步，极速合并开销。
2. **移动端适配优化**：优化虚拟终端在触控屏/手机屏幕上的呼出键盘与布局响应。
3. **真实 Android ADB 调试深度穿透**：将反向端口转发与 ADB 工具链融合，在网关后台自动绑定 5555 端口以提供开箱即用的云端 Android 调试。
